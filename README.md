# Bioconductor Funding Working Group

## Disclaimer
This repository is a prototype and is not currently officially endorsed by the Bioconductor project.

## Overview

The Bioconductor Community Advisory Board has endorsed the creation of a working group focused on funding.  
Further details about this initiative are available on the [Bioconductor Working Groups and Committees](https://workinggroups.bioconductor.org/) website.

## Objectives

This repository serves two primary purposes:

1. To compile and maintain information on current and historical sources of funding and support contributing to the Bioconductor project.
2. To highlight and document upcoming funding opportunities for which community members may collaborate on applications.
